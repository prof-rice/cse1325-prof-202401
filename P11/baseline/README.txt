# Baseline Data

File enrollment.txt includes the name and undergraduate enrollment of the largest 250 universities and colleges in the US per the National Center for Education Statistics website. 

https://nces.ed.gov/ipeds/use-the-data#SurveyData

State abbreviations were manually added based on the following table, which is also included as state_abbrev.txt for use in implementing Bonus 2.

https://www.faa.gov/air_traffic/publications/atpubs/cnt_html/appendix_a.html
